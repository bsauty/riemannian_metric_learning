################################################################################
### DATA EXPLANATION
################################################################################

This data corresponds to the cognitive scores (ADAS-Cog) of patients from the
ADNI database that converted from MCI to AD during the study (longitudinal
study with more than 2 observations per patients).

Order of the ADAS-Cog scores : memory, language, praxis, concentration




################################################################################
### CSV files
################################################################################

### group.csv
IDs of the patients. The number of lines an ID appears corresponds to the number
of observation for the given subject.

### X.csv
Age of the patients at each observation. The lines corresponds to the IDs in the
group.csv file.

### Y.csv
Values of the cognitive scores at each observation. The lines corresponds to the
IDS in the group.csv file.
